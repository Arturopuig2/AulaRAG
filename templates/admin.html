📥 Importación Masiva de Preguntas

Sube un archivo .json o .csv con preguntas para cargarlas masivamente en la base de datos. Los duplicados (por identificador) se omiten automáticamente.

📄

Arrastra aquí tu archivo JSON o CSV

o haz clic para seleccionarlo

📋 Formato esperado

JSON: array de objetos. CSV: primera fila con cabeceras. Campos:

[
  {
    "subject": "matematicas",      // obligatorio
    "grade": 2,                    // obligatorio
    "bloque": "Operaciones",       // opcional
    "contenido": "Sumas",          // opcional
    "dificultad": "normal",        // basica | normal | avanzada
    "question_type": "seleccion",  // seleccion | verdadero_falso | pasos
    "question": "¿Cuánto es 3+4?", // obligatorio
    "options": ["6", "7", "8"],    // array (JSON) o separado por | (CSV)
    "answer": "7",                 // obligatorio
    "explanation": "3+4=7",        // opcional
    "identifier": "PMAT2N0001"     // opcional, se auto-genera si falta
  }
]
